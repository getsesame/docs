## Summary

- Adds the skill-install block from the sesame monorepo's `install/install.sh` so the curl-piped installer also installs the Sesame skill.
- Runs `npx -y skills add getsesame/skills --yes --global --all </dev/null`. Previously the `-y` was consumed by npx and the `skills add` subcommand hung waiting on agent-selection input when stdin wasn't a TTY.
- Prints the manual command as fallback if npx is missing or the install fails.

## Test plan

- [x] Verified locally via `sh install/install.sh </dev/null` — installed secretctl + sesame skill across 45 agents (universal + symlinks for Claude Code, Codex, Cursor, etc.) without any prompts.
- [ ] Post-merge: confirm `curl -fsSL https://getsesame.dev/install.sh | sh` now runs the skill install leg.
